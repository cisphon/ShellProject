Example usage for the common "ls" command:

1. compile the code:   
	```bash
	g++ shell.cpp -o shell 
	```
2. run the shell:
	```bash
	./shell
	```

3. type the absolute path for the unix command you want to use:  
	```bash
	/bin/ls
	```
4. You can also put options on the commands like so: 
	```bash
   /bin/ls -l
	```
